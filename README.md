# Finance Dashboard

Modern finance dashboard UI built with React, TypeScript, Vite, Tailwind CSS, and shadcn/ui.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Features

Dashboard overview, transactions table (search, filter, sort), role-based UI (viewer / admin), insights, Recharts visualizations, Zustand state, and localStorage persistence.

Repository: [Finance-Dashborad-UI](https://github.com/hellfire20031998/Finance-Dashborad-UI) on GitHub.
